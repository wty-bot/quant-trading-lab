# Project Scaffold

This workspace is organized for the RiceQuant A-share final project.

- `src/data`: local dataset loaders and validation utilities.
- `src/strategy`: final static-data strategy implementation.
- `src/backtest`: scripts or wrappers for Max / 5-year backtests.
- `src/analysis`: result aggregation and chart generation.
- `configs`: backtest and sensitivity-test parameters.
- `reports`: generated intermediate reports, figures, and tables.
- `final_submission_assets`: final report, screenshots, and RQAlpha output copies.
- `notebooks`: exploratory analysis only.
- `tests`: data loader and strategy utility tests.
